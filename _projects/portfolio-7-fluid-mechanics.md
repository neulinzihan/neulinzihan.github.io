## Overview

Analyzed a tethered sphere system for river velocity measurement through computational fluid dynamics methods.

## Project Scope

- Derived governing equations from force balance analysis
- Computed drag coefficients across Reynolds number range
- Identified measurement limitations through parametric analysis
- Implemented solutions using MATLAB

## Key Deliverables

- Moody Chart analysis and implementation
- Parametric study of system behavior
- Documentation of measurement limitations

## Technical Skills

Fluid Dynamics, MATLAB, Data Analysis, Engineering Problem-Solving, CFD Concepts
