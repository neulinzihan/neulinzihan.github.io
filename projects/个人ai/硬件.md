## 方案一：使用现有PC/Mac（最简单）

### 最低配置
```
✅ 你目前可能已有的：
  • 笔记本/台式电脑（Windows/Mac/Linux）
  • 处理器：Intel i5/i7 或 AMD Ryzen 5/7
  • 内存：8GB RAM（可运行，但会卡）
  • 存储：256GB SSD（足够）
  • 网络：WiFi连接

⚠️ 限制：
  • 24/7运行不太现实（笔记本会发热、掉电）
  • 功耗较高（笔记本满负荷50W+）
```

### 推荐配置
```
✅ 升级建议：
  • 内存升级到 16GB（最关键）
  • 添加外接 SSD 500GB+（备份数据库）
  • 配置UPS不间断电源（最多花200-300元）

💰 成本：内存+SSD = ￥500-800 | UPS = ￥200-300
```

### 最佳配置
```
✅ 台式电脑配置：
  • CPU：Intel i7-13700K / AMD Ryzen 7 7700X
  • RAM：32GB DDR5
  • 存储：1TB NVMe SSD + 2TB HDD 备份
  • 主板：支持ECC或冗余存储
  • 电源：650W 80+ Bronze + UPS 2000VA

💰 成本：￥6000-8000（新购）
```

## 常见问题

**Q: 树莓派 4 的 4GB 和 8GB 版本怎么选？**

A: 
- 4GB：可以运行，但 SQLite + Claude 调用会有点紧张
- 8GB：推荐。多花￥200，性能稳定得多

**Q: microSD 卡容量选多少？**

A:
- 32GB：太小，系统+依赖库就快满了
- 64GB：推荐（￥50）
- 128GB：如果预算允许

**Q: 为什么推荐 USB SSD 而不是 microSD？**

A:
- microSD 卡：便宜但慢（最多100MB/s）
- USB SSD：贵一点但快 10 倍（450MB/s）+ 寿命长
- 当你的记忆数据库变大后，速度差异明显

**Q: UPS 真的必要吗？**

A:
- 如果你家里经常停电：必要
- 如果网络稳定且手动处理关机：可选
- 但建议买（￥150-250 不贵，保险费用）

**Q: 可以用旧笔记本替代树莓派吗？**

A:
- 技术上可以
- 但笔记本不适合 24/7 运行（发热、风扇噪音、影响电池）
- 树莓派只需 5W，贼省电

**Q: 数据备份怎么做？**

A:
```
方案：
  主存储：USB SSD 256GB (每天更新)
  备份 1：USB 硬盘 2TB (每周备份一次)
  备份 2：加密云存储 (可选，如 Backblaze，月￥5)
  
备份脚本：使用 cron + rsync 自动化
```

**Q: 能放在家里的角落运行吗？**

A:
```
是的！树莓派非常小巧：
  尺寸：85.6 × 56.5 × 17mm（信用卡大小）
  重量：46g
  噪音：风扇版本 <30dB（很安静）
  功耗：5W（不需要特殊冷却或电源管理）
  
可以放在：
  ✅ 书架角落
```
